# Algorithmic Trading System

A Streamlit-based algorithmic trading platform using technical indicators
and machine learning for prediction, backtesting, and risk analysis.

## Features
- SMA / EMA / RSI indicators
- Buy/Sell signals
- Strategy backtesting
- Risk metrics
- ML-based price prediction
- Trading journal

## Run App
streamlit run app.py
